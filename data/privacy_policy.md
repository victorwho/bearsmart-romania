Privacy Policy — Revox AI

Last updated: 2026-03-28

This API is used by a ChatGPT GPT to generate Revox skincare product recommendations.

What data this API receives
- selected retailer
- skin type
- priority skin concern
- optional need scores and derived skin concerns
- optional eye concern
- technical request metadata such as timestamps and server logs

What this API does not need by default
- raw selfie images
- account passwords
- payment data

How data is used
- to filter the Revox catalog
- to construct a compatible routine
- to return retailer links and explanatory notes

Storage and retention
- By default, request data should be used only to serve the response.
- Server logs may contain technical metadata required for security and debugging.
- Do not store personal data longer than necessary.

Third parties
- This API may run on Google Cloud Run.
- Users interact with the GPT through OpenAI services.
- Retailer links may direct users to third-party retailer websites.

Your choices
- Do not send information you do not want processed.
- If you later add image processing or analytics, update this policy before publishing.
